# ブラウザ最終検品

## 自動検品

```bash
npm run qa
```

ビルド、静的チェック、ChromiumによるPC幅（1440px）・スマホ幅（390px）の検品を順に実行します。主要表示、画像、横はみ出し、JavaScriptエラー、メニュー、FAQ、お問い合わせフォームを確認します。フォーム送信はテスト内で差し替えるため、Formspreeには送信されません。

PC・スマホのファーストビューと、スマホの各主要セクションを実際に画面内へ移動したスクリーンショットは `test-results/` に出力されます（検品のたびに作り直されます）。長いページを一度に撮ると、スマホ版Chromiumの画面外描画最適化によって空白が写ることがあるため、目視検品にはセクション単位の画像を使用します。

## 手動でブラウザ確認

```bash
npm run preview
```

起動後に `http://127.0.0.1:4173` をChromeまたはSafariで開きます。終了はターミナルで `Control + C` です。

Playwrightの操作を画面に表示して確認するときは次を実行します。

```bash
npm run qa:browser:headed
```
